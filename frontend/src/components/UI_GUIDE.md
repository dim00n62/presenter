## 1. WorkflowStepper - Индикатор прогресса

```
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                            │
│  ⚙️           📄           🔍           📋           ✍️           🎤      💾 │
│ Настройка  Документы    Анализ     Структура    Контент     Заметки  Экспорт│
│ Цели и     Загрузка    AI извлекает  План      Создание    Текст    Скачать│
│ аудитория   файлов      данные    презентации   слайдов  выступления  PPTX │
│   ✓          ✓           ●─────────────────────────────────────────────────│
│                          │                                                 │
│                     Текущий этап                                           │
└──────────────────────────────────────────────────────────────────────────┘

Легенда:
  ✓  - Этап завершен (зеленый, кликабельный)
  ●  - Текущий этап (зеленая обводка, пульсирующий индикатор)
  —  - Будущие этапы (серый, некликабельный)
```

## Цветовая схема

```
Основные цвета:
┌─────────────┬──────────────┬────────────────────┐
│ Компонент   │ Цвет         │ Использование      │
├─────────────┼──────────────┼────────────────────┤
│ Primary     │ Green/Teal   │ Sber, основной     │
│ Success     │ Green        │ Завершено          │
│ Info        │ Blue         │ Информация         │
│ Warning     │ Orange       │ В процессе         │
│ Danger      │ Red          │ Ошибки             │
│ AI          │ Purple/Pink  │ AI процессы        │
└─────────────┴──────────────┴────────────────────┘

Градиенты:
- from-green-50 to-teal-50   → Основной фон
- from-purple-50 to-blue-50  → AI анализ
- from-blue-50 to-purple-50  → Структура
- from-purple-50 to-pink-50  → Генерация
- from-pink-50 to-orange-50  → Заметки
- from-green-50 to-teal-50   → Экспорт
```
